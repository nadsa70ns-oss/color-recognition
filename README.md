# color-recognition
A simple OpenCV project for real-time color detection using webcam or images.

# Color Recognition with OpenCV

## Overview
A Python project that detects Red, Green, Blue, and White colors using OpenCV. It supports both webcam and image input.

## Features
- Detects multiple colors.
- Supports camera and image modes.
- Draws bounding boxes around detected objects.
- Displays the detected color name.

## Technologies
- Python
- OpenCV
- NumPy

## Runpython color_detection.py

Choose:
- camera for webcam.
- image for image detection.
